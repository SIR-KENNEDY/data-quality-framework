# ✅ Data Quality Framework for Operational Datasets

![Python](https://img.shields.io/badge/Python-3.10-blue) ![Pandas](https://img.shields.io/badge/Pandas-Validation-green)

## Overview
A reusable, rule-based data quality validation framework for operational datasets. Runs configurable checks across completeness, validity, consistency, and uniqueness — and generates a structured quality report.

## Check Categories
| Category | Examples |
|----------|---------|
| Completeness | Missing values, empty fields |
| Validity | Negative quantities, future dates, out-of-range values |
| Uniqueness | Duplicate primary keys |
| Consistency | Delivered > 2x ordered, dates in wrong order |
| Referential | Site IDs not in master list |

## How to Run
```bash
pip install -r requirements.txt
python framework/run_checks.py
```
